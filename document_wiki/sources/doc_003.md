# Еженедельный отчет по качеству поддержки премиальных клиентов

## Распаршенный текст

Документ подготовлен командой премиальной поддержки. Ответственный за отчет: Орлова Мария.
Участники разбора: Кузнецов Алексей, Сидорова Анна и команда контроля качества.

Отчет рассматривает обращения премиальных клиентов по инвестиционным продуктам, валютным переводам, банковским картам и персональному сопровождению.
Ключевые показатели: CSI премиального сегмента, NPS, среднее время ожидания ответа персонального менеджера, доля эскалаций и количество жалоб на повторное объяснение проблемы.

Основные клиентские проблемы:

- клиенты не понимают сроки обработки валютных переводов;
- по инвестиционным продуктам не хватает понятного объяснения комиссий;
- по картам премиального сегмента встречаются жалобы на лимиты операций за рубежом;
- часть клиентов повторно обращается из-за отсутствия статуса по эскалации.

Процесс контроля качества включает прослушивание обращений с эскалацией, проверку полноты консультаций и контроль обратной связи от персональных менеджеров.
Команда поддержки отдельно отмечает, что низкий CSI часто связан не с ошибкой оператора, а с отсутствием понятного статуса решения проблемы.

Рекомендации:

- добавить шаблон объяснения сроков валютных переводов;
- подготовить краткую памятку по комиссиям инвестиционных продуктов;
- настроить уведомления о статусе эскалации;
- проверять обращения с повторным объяснением проблемы как отдельный контрольный процесс.
