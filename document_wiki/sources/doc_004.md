# Обзор правил фрод-мониторинга по карточным операциям

## Распаршенный текст

Документ описывает набор правил фрод-мониторинга для карточных операций за август.
Владелец отчета: Николаев Дмитрий. Технический владелец правил: Белова Екатерина.

Рассматриваются правила:

- CARD_DENY крупная покупка после смены устройства;
- CARD_DENY нетипичная сумма в новом торговом предприятии;
- CARD_REVIEW серия операций в разных странах за короткий период;
- CARD_REVIEW операция с повышенным merchant risk score.

Основные метрики обзора: количество срабатываний, доля подтвержденного фрода, доля ложноположительных срабатываний, среднее время ручной проверки и количество клиентских обращений после блокировки.

Отдельно описан процесс контроля качества правил. Аналитики проверяют выборку ложноположительных блокировок, сравнивают комментарии операторов и передают владельцам правил предложения по изменению порогов.

Клиентские проблемы в документе связаны с недостаточно понятными уведомлениями о блокировке, задержкой ручной проверки и повторными обращениями по одной операции.

Рекомендации:

- снизить чувствительность правила для постоянных клиентов с подтвержденной историей операций;
- добавить понятный текст уведомления о причине дополнительной проверки;
- выделить отдельный мониторинг ложноположительных срабатываний по premium-картам;
- добавить weekly-review правил с высокой долей клиентских обращений.
