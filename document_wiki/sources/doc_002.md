# Отчет по клиентским обращениям за июль

## Распаршенный текст

Отчет описывает динамику клиентских обращений за июль. Автор отчета: Иванов Иван.
Владелец блока по клиентскому опыту: Сидорова Анна.

Основные продукты в отчете: банковские карты, переводы, мобильное приложение и интернет-банк.
Ключевые метрики: CSI, NPS, количество обращений, доля повторных обращений, среднее время первого ответа и доля обращений, решенных с первого контакта.

По банковским картам выросло число обращений по спорным списаниям и отклоненным операциям в зарубежных интернет-магазинах.
Клиенты чаще всего жалуются на недостаточно понятное объяснение причины отказа операции.

По переводам зафиксированы обращения о задержках межбанковских переводов и некорректном отображении комиссии до подтверждения операции.
По мобильному приложению основные жалобы связаны с ошибками авторизации после обновления версии приложения и задержкой push-уведомлений.

Команда контроля качества прослушивает обращения с низким CSI, проверяет корректность консультаций операторов и передает повторяющиеся проблемы продуктовым владельцам.
Для обращений с повторным контактом формируется отдельный список причин эскалации.

В июле отдельно анализировались ложноположительные срабатывания фрод-мониторинга по переводам новым получателям.
В отчете рекомендовано улучшить тексты уведомлений о дополнительной проверке операций и добавить объяснение сроков проверки.

Рекомендации:

- обновить тексты отказов по карточным операциям;
- показать комиссию по переводу до подтверждения операции;
- усилить контроль обращений с повторным контактом;
- выделить отдельный weekly-review по ложноположительным фрод-срабатываниям;
- передать команде мобильного приложения список ошибок авторизации после обновления.
