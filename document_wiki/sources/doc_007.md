# План обучения операторов по сложным клиентским обращениям

## Распаршенный текст

Документ подготовлен учебным центром клиентской поддержки. Автор: Федорова Наталья.
Ответственные подразделения: учебный центр, команда контроля качества, руководители линий поддержки.

Цель плана — повысить качество консультаций по сложным клиентским обращениям и снизить долю повторных контактов.

Темы обучения:

- объяснение причин отклонения карточных операций;
- консультации по задержкам переводов;
- работа с обращениями по фрод-блокировкам;
- объяснение статуса эскалации;
- коммуникация с премиальными клиентами;
- корректное заполнение причины обращения в CRM.

Метрики эффективности обучения:

- результат тестирования операторов;
- доля успешных контрольных звонков;
- CSI после консультации;
- доля повторных обращений;
- количество ошибок в классификации причин обращения;
- среднее время консультации по сложным сценариям.

Процесс контроля включает тестирование до и после обучения, выборочную проверку диалогов, контроль заполнения CRM и разбор ошибок на еженедельных встречах руководителей групп.

Рекомендации:

- создать отдельный трек обучения по фрод-блокировкам;
- добавить практические кейсы по спорным списаниям;
- включить в обучение сценарии объяснения сроков валютных переводов;
- связать результаты обучения с метриками CSI и повторных обращений;
- использовать ошибки классификации в CRM как материал для следующих обучающих модулей.
